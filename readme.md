# Container Demos

## Introduction

Here are some examples of playing around with containers, cgroups, and namespaces.  These
examples/demos give some hands-on experience around some the DrupalCon Austin 2014 session.
See the acommpanying slides at www.slideshare.net/GetPantheon/drupalcon-2014-mo-servers-mo-problems.


## How to use
* `vagrant up` and try yourself
* Check out the examples
